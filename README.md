# Auto Buy Wallet Bot

## 🚀 How to Run

### Method 1 (Console)
1. Open target website
2. Open DevTools (F12)
3. Paste:

var s=document.createElement('script');
s.src="https://raw.githubusercontent.com/mrkayastharahul-cell/Autobuyarwalletbyrahul/main/main.js";
document.body.appendChild(s);

4. Run:

startAutoBuy("1000", 400)

---

## ⚡ Parameters

startAutoBuy(amount, speed)

- amount → number to match (e.g. "1000")
- speed → delay in ms (200 = fast, 500 = normal, 1000 = slow)

---

## 🔥 Features

- Auto click Default
- Filter exact amount
- Auto click Buy
- Detect payment page
- Auto select MobiKwik
- Sound alerts
